Apply fees based on the WooCommerce payment gateway selected by the customer.

== Description ==

PayFeez extends WooCommerce by adding options for setting fees based on the payment gateway selected by the customer.

Payment gateway fees can be added to many payment gateways, including :
* PayPal
* Stripe
* Stancer

PayFeez also gives you the option of restricting orders below a certain purchase amount.
